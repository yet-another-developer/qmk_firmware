# User Space for yet-another-developer

## Reset keyboard
LOWER + KC_LBRC (hold space + [)

## Flashing

Flash
```
cd ~/qmk_firmware
make ergodash/rev1:yet-another-developer:flash
```

Generate build without flashing
```
cd ~/qmk_firmware
make ergodash/rev1:yet-another-developer
```
